Little Alchemy : Fun Element Mixing Game | GitHub Games

Little Alchemy is a captivating puzzle game where players combine basic elements like fire, water, earth, and air to discover new items. With hundreds of possible combinations, it's perfect for creative minds!

How to Play Little Alchemy:

  1. Start with 4 basic elements: air, earth, fire, and water.
  2. Drag and drop elements onto each other to create new items.
  3. Discover all 580+ possible combinations (including hidden ones!).
  4. Unlock advanced items like dinosaurs, robots, and even entire universes!

Pro Tip: Try combining unexpected elements - sometimes the most unlikely pairs create amazing results!
